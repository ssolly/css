제목 공간

박스모델 padding, border, margin 속성의 기본값은 "0"
상우하좌 방향(top,right,bottom, left)

[border]
- 숫자 하나 작성 시 : 모든 방향에서 동일 값 적용
- border-top/bottom/right : 개별적 박스 적용
- solid / dotted / double : 테두리 모양 설정
- ex) border-bottom:15px double gray; border-right:5px dotted pink;

[padding]
- 안쪽으로 여백을 주는 것
- padding-top/bottom : 개별적 안쪽 여백 적용
- padding값 두개만 작성시 : 상하 / 좌우 여백 적용
- ex) padding:20px 0;

[margin]
- 바깥으로 밀어내는 것
